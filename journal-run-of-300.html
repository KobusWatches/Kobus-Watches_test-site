A run of 300

On starting with a small production run, and what it takes to scale from there.

The first production run will be 300 pieces. The number is small on purpose.

At 300, every case can be inspected individually. Suppliers can be visited rather than emailed. If a finishing step is not right, the run is small enough to stop, correct, and restart without sinking the project.

Scaling comes later, and only as far as quality allows. A second run is only worth doing if it is indistinguishable from the first — except in the lessons applied.
